Welcome to the 'Self-Hosted DeepSeek R1' VSCode extension for interacting with your locally hosted DeepSeek AI. This is an unofficial extension.

Getting Started

1. Configure the extension, press CTRL + , then search for DeepSeek.. OR use the command CTRL + SHIFT + P and type DeepSeek Config

2. Open DeepSeek in the Sidebar CTRL + SHIFT + P and type Show DeepSeek to open the default chat panel.

OR. Open a floating window DeepSeek Chat, use command CTRL + SHIFT + P and type DeepSeek Floating Chat to open a Chat as a floating window.

OR. Ask DeepSeek about selected Code, Highlight a block of code in your editor, righ click and choose Ask DeepSeek. Enter your question and use enter key to send.

Hosting DeepSeek on Ollama

Ollama is a lightweight self-hosting solution for running AI models locally. For setup instructions: How to run DeepSeek Locally on Ollama.
